# Ledger Simulator

A simple Kotlin-based ledger simulator for tracking transactions and balances. Built with Gradle and tested using JUnit with JaCoCo.

## Features

- Add, edit, and remove transactions
- Track balances for multiple accounts
- Unit tested with coverage reports

## Getting Started

### Prerequisites

- Java 17+  
- Gradle 8+  
- Git  

### Build & Run


git clone https://github.com/RominaVojnovski/ledger-simulator.git
cd ledger-simulator
./gradlew build
./gradlew run
